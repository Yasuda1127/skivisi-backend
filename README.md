# プロジェクトの概要
エンジニアスキル可視化サイト。<br />
ユーザー登録からスキルのレーダーチャートの表示、スペックシートの提出・承認ができるシステムを構築。

URL：https://skivisi-3obfpgmab-yasuda1127.vercel.app/


# 使用方法、インストール方法、実行方法
- パッケージインストール
```
npm install
```

- サーバー起動
```
npm run dev
```
